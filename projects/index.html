Health & Fitness Data Pipeline

Power BI Python Azure Tables

The Problem: Managing personal physiological data (sleep, fasting cycles, heart rate) across disparate APIs restricts a unified view of biological progression and recovery.

The Approach: Developed automated Python scripts that pull raw JSON metrics directly from the Fitbit API. These pipelines transform unstructured biological logs into structured historical tables and Azure Blob layers. I then built dynamic Power BI dashboards on top, exposing nuanced, multi-layered correlations.

The Outcome: Established an automated daily "health intelligence" brief. The framework is highly extensible and can be repurposed for broader community wellness tracking.
